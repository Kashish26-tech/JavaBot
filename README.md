# 🤖 JavaBot — AI-Powered Desktop Chatbot

A Java desktop chatbot powered by Groq AI (Llama 3.1) with a JavaFX GUI.
Built from scratch as a beginner Java project.

![Java](https://img.shields.io/badge/Java-17%2B-orange?style=for-the-badge&logo=java)
![JavaFX](https://img.shields.io/badge/JavaFX-26-blue?style=for-the-badge)
![Groq](https://img.shields.io/badge/Groq-Llama%203.1-green?style=for-the-badge)

---

## ✨ Features
- 🤖 Real AI replies powered by Groq (Llama 3.1)
- 🌙 Dark / Light mode toggle
- 🧠 Remembers your name
- 💾 Auto-saves chat history to a file
- 🧮 Simple math calculator
- 🚀 Silent double-click launcher

---

## ⚙️ Setup
1. Install JDK 17+ and JavaFX 26
2. Get a free Groq API key from https://console.groq.com
3. Add your key to `GroqAI.java`
4. Run `run.bat` to compile and launch
